Nomenclatura dos arquivos

Aviso: Este artigo foi atualizado em 17/08/2016, e só é válido para o desenvolvimento do AVA.

A nomenclatura dos arquivos desenvolvidos deve ser seguida à risca, as especificações estão a seguir:

  • O início deve ser "ilustra", "animacao", ou "SA" dependendo do tipo de conteúdo
  • Após isso não deve ter espaçamento, e deve ser seguido do número do tipo de conteúdo com 2 dígitos, por exemplo: 08, 37, 12, ...

Observe a seguir exemplos de nomenclatura:

animacao03

ilustra09

SA23

Obs: Passe a seta por cima dos elementos acima para ler a descrição.

Além disso, para cada curso deve conter pastas com os tipos de arquivos semelhantes, veja a seguir:

  • Pasta AVA: Arquivos referentes ao padrão scorm, onde estará a última versão do curso.
  • Pasta Fontes: Pasta onde terão todos os arquivos fontes.
  • Pasta Fontes > Animacoes: Pasta com as animações.
  • Pasta Fontes > Videos: Pasta com os vídeos.
  • Pasta Fontes > Simuladores: Pasta com os simuladores.
  • Pasta Fontes > Imagens: Pasta com as imagens (incluso infográficos).
  • Além disso, para as animações, vídeos e simuladores, é permitido criar pastas para cada um deles, desde que siga a nomenclatura padrão.
