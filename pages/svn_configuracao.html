Configuração das pastas

Antes de tudo deve ser feito a configuração inicial das pastas do SVN (chamamos de repositórios). Para fazer isso, primeiramente você deve ter o programa TortoiseSVN instalado. Peça para o suporte instalar o software, para agilizar o processo já faça o download do software para que eles possam instalá-lo: http://bit.ly/200Z3Jl

Criação das pastas

Após isso, deve ser criado as pastas, isso é feito normalmente como sempre fazemos (botão direito - Novo - Pasta), mas recomendamos criar com algumas regras. Primeiramente crie uma pasta no seu D: (abrindo o Meu computador, também pode ser que não exista um D:, as vezes pode ser que seja outra letra, como E:, ou tenha somente o C:) chamado de "Desenvolvimento AVA" e outra chamada "Desenvolvimento LD".

Após a criar estas pastas (ou somente as que você precisa, para poupar tempo) deverá ser feito o checkout das mesmas. Para fazer isso, você deve clicar com o botão direito na pasta correspondente e clicar na opção SVN Checkout..., como mostra a figura a seguir:

Após isso, abrirá uma janela para fazer o Checkout do repositório. No primeiro campo de texto você insere o endereço das pastas. Caso queira ver todos os links disponível, acesse o menu SVN desta documentação e em seguida em Endereços dos projetos ou clique aqui.

Após colocar o endereço no campo correto, como mostra a figura a seguir, clique em OK nesta janela. Agora ele começará a buscar todos os arquivos do servidor deste curso para o seu computador, dependendo do tamanho do projeto este processo poderá demorar um pouco. Assim que terminar, clique em OK para fechar a janela e estará pronto a configuração desta pasta. Você deve fazer este processo para cada pasta que deseja trabalhar com seu endereço correspondente.
