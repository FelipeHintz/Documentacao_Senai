Criando os arquivos

Após ter sido feito o checkout, você já pode começar a colocar os arquivos que você está trabalhando nas devidas pastas. Enquanto você estiver trabalhando nos arquivos não é necessário nenhuma comando ou configuração, mas assim que você terminar, será necessário executar o comando commit. Para fazer isso, clique com o botão direito na pasta do curso e clique na opção SVN Commit... como mostra a figura a seguir.

Após clicar em SVN Commit..., abrirá uma tela como a imagem a seguir. A partir desta tela é que você vai enviar os arquivos para o servidor. No campo de texto acima você pode escrever uma mensagem referente ao que você está fazendo, por exemplo: "Desenvolvido vídeos do slide 123". Já abaixo terá uma lista dos arquivos que ainda não estão no servidor. Estes são os arquivos que você deve selecionar na caixa ao lado esquerdo, para agilizar o processo, existe um link logo acima destes arquivos escrito All que selecionará todos os arquivos abaixo.

Após isso, você pode clicar no botão OK. E ele começará a enviar os arquivos para o servidor, com uma tela semelhante a imagem a seguir.

Assim que terminar (quando liberar o botão OK) você pode fechar, ou clicar em OK.
