Destaque

Detalhamento

Aplicação de uma área de texto que necessite de destaque. O recurso foi disponibilizado de duas formas: sem ícone, aplicando apenas a classe destaque, ou com ícone, aplicando a classe destaque juntamente com uma das seguintes classes: animacao, atividade, casosErelatos, fiqueAlerta, recapitulando, saibaMais e voceSabia. Cada classe aplica o ícone correspondente seguindo o padrão de cores do curso.

Existe três formas de adicionar links nas caixas, a primeira é colocar um link no conteúdo, a segunda é trocar o div externo por um link e a terceira é colocar o parâmetro onclick no div externo, com style="cursor:pointer", como pode ser observado no código a seguir.

Código - HTML

Light Box

Este é mais um recurso que possibilita dar destaque em uma parte do texto, porém com um recurso mais dinâmico. Nele o aluno clica sob a imagem e abrirá uma janela por cima do conteúdo apresentando a imagem com o texto em evidência. Exemplo ilustrado na imagem seguinte.

Detalhamento

Para este recurso, são necessárias duas imagens:

  • uma que será inserida na TAG IMG que aparecerá no corpo de texto;
  • e outra que referenciada na TAG A será imagem apresentada no efeito Java Script, como no exemplo ilustrado anteriormente;
  • como mostrado no primeira ilustração existe uma imagem de um ícone ao lado da imagem de uma “mãozinha”, isso significa que o aluno deve clicar sobre a imagem, para isso o ícone deve ser inserido ao lado da própria imagem;
  • note que o texto da light box está sendo inserido dentro da TAG IMG no atributo ALT, isso é necessário para que o software para deficientes consiga ler as informações para o aluno;
  • para exibir um caption com a imagem é necessário colocar o texto no parâmetro title do link.

Código - HTML 
<style>
.fancybox-image { margin-top:-30px; }
</style>

Texto sobre Imagem

Este recurso possibilita uma ligação entre texto e imagem. Ao passar o mouse sob a imagem, o aluno terá acesso a um texto que aparecerá por cima da ilustração. Veja, na figura a seguir, o exemplo do texto após ter posicionado o mouse em cima da imagem.

Detalhamento

Para que o recurso funcione corretamente e que o software para deficientes consiga ler as informações, você deve colocar o texto dentro da TAG SPAN.

O ícone localizado no canto direito superior é apresentado automaticamente pelo plugin, com a cor de fundo bg_main definida no skin_vet.

Código - HTML 
<img class="txtOverImg" src="imgs/04.05.02.png" alt="" name="#txtOverImg2">
<div id="txtOverImg2">
  <span>O primeiro curso t&eacute;cnico de Fisioterapia, com dura&ccedil;&atilde;o de 1 ano,
  foi criado em 1951 na Universidade de S&atilde;o Paulo.</span>
</div>
Especificações

Formato: PNG ou JPG
Tamanho máximo: 300x300

Imagem

Detalhamento

Imagem fixa no canto direito deixando o texto correr ao redor. A rotação, a sombra e a borda são inseridas automaticamente pela classe figuraMoldura.

Você deve inserir a descrição da imagem no atributo ALT da tag IMG, para que o software para deficientes consiga ler as informações.

Código - HTML 
<img src="imgs/04.05.02.png" class="figuraMoldura" style="width:200px">
<p>Mediante a situa&ccedil;&atilde;o  contextualizada anteriormente, voc&ecirc; decide ent&atilde;o buscar se informar sobre o  assunto, e inicialmente assiste ao v&iacute;deo &ldquo;O caso da Escola Clarisse Fecury&rdquo; &ndash;  Rio Branco, AC - Dura&ccedil;&atilde;o 6:34. Que voc&ecirc; pode ter acesso.</p>
Especificações

Formato: PNG ou JPG
Tamanho máximo: 300x300

Zoom

Detalhamento

Recurso utilizado para imagens que exijam uma visão mais detalhada.

Para que o efeito funcione perfeitamente é preciso ajustar o width e height na tag IMG na linha 6, para definir o tamanho de exibição na tela. No exemplo acima foi utilizada uma imagem de 704x553, e foi informado 310x177, para possibilitar o efeito de zoom.

O icone localizado no canto direito superior é apresentado automaticamente pelo plugin, com a cor de fundo bg_main definida no skin_vet.

Você deve inserir a descrição da imagem dentro da última TAG SPAN que aparece com a classe legenda e dentro da TAG IMG no atributo ALT ilustrada na imagem acima para que o software para deficientes consiga ler as informações.

Código - HTML 
<span class="zoom">
	<img src="imgs/car.jpg" width="263" height="150" alt="Erros de linguagem dos alunos">
</span>

<span class="zoom">
	<img src="imgs/04.05.02.png" width="150" height="150" alt="Outra imagem com zoom">
</span>
<span class="legenda">Erros de linguagem dos alunos</span>
Especificações

Formato: JPG e PNG
Tamanho máximo: 350x350

A imagem que será mostrada na tela de conteúdo é inserida no link que abrirá a galeria. Já as imagens que farão parte da galeria devem ser inseridas no script.

Vídeo

Detalhamento

Os vídeos podem ser inseridos no conteúdo individualmente ou em um grupo (playlist). Em ambos casos é necessário informar as dimensões do vídeo no parâmetro style do link e o caminho do arquivo no parâmetro href, como mostrado no código HTML acima. No vídeo único, o parâmetro id deve ser único, e iniciar com video_, já para a playlist, o id do link, também único, deve iniciar com videos_ e o id do div que engloba os vídeos deve iniciar com playlist_, sendo que o restante do id após estes prefixos deve ser igual na mesma playlist. Este padrão é necessário para que os vídeos sejam inseridos automaticamente.

Sobre o tamanho dos arquivos de vídeo, o ideal é um tamanho máximo de 20MB, caso haja uma grande necessidade de um vídeo maior, pode ser inserido em casos esporádicos, tendo em vista o tamanho do pacote final SCORM.

Código - HTML 
<h1>V&iacute;deo &Uacute;nico</h1>
<div style="width:720px;height:480px;margin:20px auto">
  <a id="video_1" class="player" href="video/video.mp4" style="width:720px;height:480px"></a>
</div>

<h1>Playlist de V&iacute;deos</h1>
<div class="playlists">
  <a id="videos_1" class="player" href="" style="display:block;width:320px;height:240px"></a>
  <div id="playlist_1" class="playlist">
    <a href="video/uc2_ue1_01.flv"><strong>Mult&iacute;metro</strong></a>
    <a href="video/video.mp4"><strong>V&iacute;deo</strong></a>
  </div>
</div>
Especificações

Formato: MP4, F4V e FLV
Resolução máxima: 600x400
Tamanho máximo de arquivo: 20MB

Flash (SWF)

Em caso da necessidade de inserção de animações SWF.

É importante lembrar que no novo template os recursos em Flash devem ser evitados. Só devem ser utilizados se não houver possibilidade de substituição por outra tecnologia que funcione em uma ampla gama de dispositivos (de desktops a dispositivos móveis).
Em caso de utilização deste software, o aluno não pode ter seus estudos prejudicados caso acesse o curso por meio de um dispositivo móvel.
Detalhamento

As animações em flash são inseridas no conteúdo automaticamente, mas para isso é necessário que o parâmetro id do div que a engloba seja único e inicie com flash_. Também é necessário informar as dimensões no parâmetro style do div e o caminho do arquivo no conteúdo do mesmo, como mostrado no código HTML abaixo.

Código - HTML 
<div id="flash_1" style="width:640px;height:460px;margin:0 auto">anim/animacao2.swf</div>

Pop Up

Efeito ativado.

Detalhamento

Para este recurso, são necessários dois arquivos: um será uma imagem inserida na TAG IMG que aparecerá no corpo de texto; e o outro uma imagem que referenciada na TAG A o SWF que será apresentada no efeito Java Script, como no exemplo ilustrado anteriormente.

Como mostrado no primeira ilustração, existe uma imagem de um ícone ao lado da imagem de uma “mãozinha”, isso significa que o aluno deve clicar sobre a imagem e por isso o ícone deve ser inserido ao lado da própria imagem.

Note que o texto da light box está sendo inserido dentro da TAG IMG no atributo ALT, isso é necessário para que o software para deficientes consiga ler as informações para o aluno.

Confira o código-fonte abaixo.

Código - HTML 
<a class="fancybox" href="anim/contextualizacao.swf">
  <img src="anim/img_cont.png" alt="Que em 1854, no Hosp&iacute;cio D. Pedro II dist&uacute;rbios psiqui&aacute;tricos?">
</a>
Especificações

Formato: SWF
Resolução máxima: 600x400
Tamanho máximo de arquivo: 10MB

Glossário/Informações adicionais

Este recurso tem a finalidade de trazer informações adicionais.

Detalhamento

Observando o código abaixo você irá perceber que o efeito é ativado pela função overlay, no parâmetro href do link. Essa função precisa de dois parâmetros: sendo o primeiro o título e o segundo o texto do pop up.

A outra forma é criar um link, div ou span com o ID iniciando com bt_hint_, e o conteúdo do hint fica em um div com ID iniciando com hint_ e o mesmo sufixo. Esse elemento deve possuir display none, o título no parâmetro title e a mensagem HTML a ser exibida no conteúdo.

Como você pôde observar, os pop ups devem seguir a cor da área do curso em questão.
Código - HTML

Especialistas de diversos pa&iacute;ses trabalharam  como consultores, contribuindo para consolidar sua fama de excel&ecirc;ncia, que  atraiu pacientes do Brasil e tamb&eacute;m de outros pa&iacute;ses da Am&eacute;rica Latina.</p>

Pop Up Externo

Este recurso é utilizado para abrir uma nova janela do navegador que apresenta uma identidade visual padrão, conforme a próxima ilustração.

Detalhamento

Esta funcionalidade pode ser utilizada para inserção de animações, simuladores ou texto. Para que o recurso funcione, você deverá criar uma página html com o conteúdo que irá aparecer no pop up. A abertura do conteúdo externo é feita através da charmada da função popup, ilustrada na imagem abaixo, passando no primeiro parâmetro o caminho do arquivo HTML, e no segundo as dimensões da janela, sendo w a largura e h a altura.

Como você pôde observar, os pop ups devem seguir a cor da área do curso em questão, assim como o nome do curso e logomarca do SENAI.
Lembrando que a logomarca do SENAI pode ser utilizada na forma negativa ou positiva, esta decisão deve ser tomada mediante harmonia com a área do curso.
Código - HTML 
<p><a href="javascript:popup('popUp.html',{w:625,h:460})">Especialistas de diversos pa&iacute;ses trabalharam como consultores.</a></p>
